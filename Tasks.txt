Student registration app

Features:

    Signup & Login (Home page, "/")

    Dashboard

    registration

    FRONT END:

    Signup & Login:

        Signup (student):
            - Input for name, email, and password.
            - Create Account button that sends a request to the backend & inserts the student info into the database (students tables).
            - Route to login page (if student has an account).

        Login (Student):
            - Input for student email and password.
            - Login button that takes you to student Dashboard.
            - Route to signup page (if student has not signed up).
